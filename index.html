CSS Selectors

CSS selectors are used to select the HTML elements you want to style. Here are some common types of selectors:

  • Class: Selects elements with a specific class attribute. Example: .classname
  • ID: Selects a single element with a specific ID attribute. Example: #idname
  • Tag: Selects all elements of a given type. Example: p for all paragraph elements.
  • Attribute: Selects elements with a specific attribute. Example: [type="text"]
  • Universal: Selects all elements. Example: *
  • Descendant: Selects elements that are descendants of another element. Example: div p
  • Child: Selects elements that are direct children of another element. Example: div > p
  • Adjacent Sibling: Selects an element that is immediately preceded by another element. Example: h1 + p
  • General Sibling: Selects all elements that are preceded by a specified element. Example: h1 ~ p

Colors in CSS

Colors in CSS can be specified using predefined color names, HEX values, RGB, RGBA, HSL, and HSLA values. Here are some examples:

  • Color Names: red, blue, green
  • HEX Values*:Hexadecimal color values #ff0000, #0000ff, #008000
  • RGB Values*:Red, Green, Blue color values rgb(255, 0, 0), rgb(0, 0, 255), rgb(0, 128, 0)
  • RGBA Values*:RGB with Alpha (opacity) rgba(255, 0, 0, 0.5), rgba(0, 0, 255, 0.5), rgba(0, 128, 0, 0.5)
  • HSL Values*:Hue, Saturation, Lightness hsl(0, 100%, 50%), hsl(240, 100%, 50%), hsl(120, 100%, 25%)
  • HSLA Values*:HSL with Alpha (opacity) hsla(0, 100%, 50%, 0.5), hsla(240, 100%, 50%, 0.5), hsla(120, 100%, 25%, 0.5)
